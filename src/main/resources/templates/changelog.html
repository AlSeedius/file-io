Новая версия 0.85!

Новые функции:

  1. Новая вкладка "График". Отображает привычный график, аналогичный бумажному варианту. Перемещение по графику осуществляется с помощью кнопок в верхней части экрана, либо с помощью свайпов вверх/вниз. Число отображаемых дней определяется в настройках.
  2. Виджет "Календарь". После установки новой версии в списке виджетов Вашего устройства добавится календарь с графиком дежурств, аналогичным соответствующей вкладке. Для обновления информации на виджете необходимо нажать по нему.
  3. Обновление, а не переустановка. Теперь нет необходимости удалять приложение перед установкой новой версии - проблем с уведомлениями не будет при простом обновлении

Исправления:

  1. Исправлено отображение главной страницы у специалистов со скользящим графиком.
  2. Применены небольшие исправления, улучшающие общую стабильность работы
